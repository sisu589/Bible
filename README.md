# 📖 Bible Reading Repository

A clean, daily-driven repo for reading the Bible chapter by chapter, practicing typing, GitHub, and reflection.
## Berean Study Bible (BSB)
## Daily Workflow
1. Every day at the same time, create a new Markdown file named like this:  
   `Genesis.1.md` (Book.Chapter.md)
2. Copy the full chapter text into the file
3. Add the file to GitHub (or use the optional auto-commit script below)
4. Commit with a short message like:  
   `Added Genesis.1`
5. Push to GitHub

## Files in the Repo
- Every chapter is stored as its own `.md` file (e.g., `Genesis.1.md`, `Exodus.1.md`, `John.3.md`, `Revelation.22.md`)
- No other files are needed

## How to Use This Repo
- **Read the Bible**: Just open any chapter file on GitHub. Beautiful, consistent, and offline-friendly.
- **Practice Typing**: Type each chapter yourself (great for muscle memory and focus).
- **Practice GitHub**: Every day is a real Git commit + push.
- **Reflect**: Keep a small note at the end of each chapter file (see example below).
